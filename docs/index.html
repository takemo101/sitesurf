SiteSurf

AIと一緒にWebを操作しよう。チャットで指示するだけで、ページの読み取り・操作・ナビゲーションを自動化するChrome拡張機能です。

ダウンロード GitHub

SiteSurf とは？

📄

ページ読み取り

現在のページのテキスト・DOM構造をAIに送信し、内容を理解させます。

🔧

DOM操作

AIがJavaScriptでページ要素を操作。クリック、入力、データ抽出を自動化。

🧭

ナビゲーション

AIが指定URLへ自動移動。複数ページにまたがるタスクもこなします。

🎯

要素選択

ページ上の要素をクリックして選択し、AIに直接渡せます。

📸

スクリーンショット

ページのスクリーンショットをAIに送信。視覚的な情報も活用できます。

🤖

マルチプロバイダー

Claude, ChatGPT, Gemini, Copilot, Ollama など多数のAIに対応。

プロバイダー 認証方式
Anthropic (Claude) APIキー
OpenAI (ChatGPT) APIキー / OAuth

インストール方法

1

Chrome バージョンの確認

Google Chrome の最新版がインストールされていることを確認してください。

2

SiteSurf をダウンロード

GitHub Releases から最新リリースの dist.zip をダウンロードしてください。

GitHub Releases からダウンロード
3

ZIP を展開

ダウンロードした ZIP ファイルをわかりやすい場所に展開します。

  • Windows: C:\Users\YourName\Desktop\sitesurf
  • macOS: ~/Desktop/sitesurf
4

Chrome 拡張機能ページを開く

アドレスバーに chrome://extensions と入力して Enter を押します。

5

デベロッパーモードを有効化

右上の「デベロッパー モード」トグルをオンにします。

6

拡張機能を読み込む

「パッケージ化されていない拡張機能を読み込む」をクリックし、展開した sitesurf フォルダを選択します。

7

拡張機能の設定

chrome://extensions で SiteSurf の「詳細」をクリックし、以下を有効にします。

  • ユーザー スクリプトを許可する — AIが生成したコードをページ上で実行するために必要です
8

SiteSurf を起動

Chrome 右上の SiteSurf アイコンをクリック、またはキーボードショートカットで開きます。

  • macOS: Cmd + Shift + E
  • Windows: Ctrl + Shift + E
9

AI プロバイダーを設定

設定画面で使用するAIプロバイダーを選択し、APIキーまたはOAuth認証を設定します。

アップデート方法

1

最新版をダウンロード

GitHub Releases から最新の dist.zip をダウンロードします。

最新版をダウンロード
2

ファイルを差し替え

既存の sitesurf フォルダの中身を新しいファイルで置き換えます。

3

拡張機能をリロード

chrome://extensions で SiteSurf カードの更新ボタンをクリック、または Chrome を再起動します。

使い始める

SiteSurf を開いたら、テキストボックスにやりたいことを入力して送信するだけ。AIがページを理解し、操作を実行します。

使用例:

  • 「このページのすべてのリンクを一覧にして」
  • 「検索ボックスに"Chrome拡張"と入力して検索ボタンを押して」
  • 「このページの価格情報を抽出して」

Note: SiteSurf は開発中のソフトウェアです。問題が発生した場合は Escape キーまたは停止ボタンで即座に停止できます。

プライバシー & セキュリティ

データはあなたの手元に

設定・APIキー: ブラウザの IndexedDB にローカル保存。使用するプロバイダー以外には送信されません。

チャット履歴: すべてローカルに保存されます。

メッセージ送信時: 選択したAIプロバイダーにのみ送信されます。
