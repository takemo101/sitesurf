## 概要

Mantine is a React components and hooks library with native dark theme support and focus on usability, accessibility and developer experience. This organized documentation provides a comprehensive guide to using Mantine in your projects.

## 利用方法

このドキュメントはAIエージェント向けに構成されており、以下の特徴があります:

- **構造化されたカテゴリ** - 関連するコンテンツが論理的にグループ化されています
- **適切なファイルサイズ** - 各ファイルは500〜3000行に調整されており、AIのコンテキストウィンドウに収まります

- **ナビゲーション要素削除** - サイドバー、フッター、広告は削除されています
