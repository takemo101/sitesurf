## Mantine Core components

[@mantine/core](https://www.npmjs.com/package/@mantine/core) package provides essential components: buttons, inputs, modals, typography, and many others. The `@mantine/core` package is used internally in most of the other `@mantine/*` packages – it is required to be installed in your project to use Mantine components.
