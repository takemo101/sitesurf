## Prerequisites

Before getting started, you need:

1. A Node.js environment (v18 or higher)
2. An API key from an AI provider (OpenAI, Anthropic, etc.)
3. Basic knowledge of your chosen framework

## Installation

All guides start with:

```bash
npm install ai
```

Then configure your provider with the appropriate API keys.

## Common Steps

1. **Choose a Provider** (see Choosing a Provider guide)
2. **Install AI SDK** (`npm install ai`)
3. **Create API route/endpoint** (framework-specific)
4. **Set up environment variables** (API keys)
5. **Call the AI SDK functions** (see Core documentation)
6. **Handle responses** (text, streaming, or structured data)

## Next Steps

After choosing your framework:

1. Install and configure
2. Review framework-specific examples
3. Read AI SDK Core for detailed API reference
4. Check cookbook for real-world examples
